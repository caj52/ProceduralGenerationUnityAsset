This is a simple Unity Editor tool that gives you access to a variety of complex and performant noise generation functions, 
as well as a graphical editor where you can customize and visualize the noise as well as export its gen settings for use in your code base.

I made this as a sort of one size fits all solution for visualizing and editing noise for projects that utilize procedural generation.
Debugging the noise and layers and blending and all the different things we use procedural generation to create can become very esoteric, 
so I created this asset to simplify the processes and make it easier to create interesting, unique noise.
I never finished it but it has a very solid set of base functionalities. Feel free to use it however you'd like.
